# SarasLabs Website

Free GitHub Pages website for SarasLabs and SARAS ERP.

## Upload Steps
1. Open your GitHub repository: `saraslabs-website`
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Source: Deploy from branch.
5. Branch: `main` and folder `/root`.
6. Save.

Your website will be live at:
`https://YOUR_USERNAME.github.io/saraslabs-website/`

Note: If GitHub rejects the `.exe`, upload it in GitHub Releases and replace the download link in `index.html`.


Note: Setup .exe is not included because GitHub web upload has a 25 MB limit. Upload setup to Google Drive or GitHub Releases and update the download button link in index.html.
